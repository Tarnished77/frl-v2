# Locked Publication Figures

These figures are generated by `research/frl_v2/plot_results.py` from the
tracked compact result layer. Each figure has a vector PDF and a 450 dpi PNG
fallback. `figure_manifest.json` records the source commit, result-manifest
hash, and file checksums.

An independent replay produced byte-identical PDF and PNG files. All PDF fonts
are embedded and subsetted. Main figures are numbered 2--3 because the editable
conceptual model diagram is Figure 1; Supplementary figures are continuously
numbered S1--S7.
